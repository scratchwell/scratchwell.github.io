TouchPilot 개인정보 처리방침

시행일: 2026년 6월 23일

TouchPilot은 ScratchWell이 제공합니다. 이 문서는 TouchPilot을 사용할 때 정보가 어떻게 처리되는지 설명합니다.

요약

TouchPilot은 TouchPilot 계정, 개발자 서버, 광고 SDK, 분석 SDK 또는 추적 기능을 사용하지 않습니다. 이 앱은 iPhone에서 본인의 iPad로 키보드와 포인터 입력을 Bluetooth로 보내도록 설계되었습니다.

수집하지 않는 정보

TouchPilot은 다음 정보를 수집, 저장, 판매 또는 공유하지 않습니다.

  • 이름, 이메일 주소 또는 계정 정보
  • ScratchWell 서버로 전송되는 입력 텍스트 또는 키 이벤트
  • ScratchWell 서버로 전송되는 Bluetooth 기기 목록
  • 사용 분석 정보
  • 광고 식별자
  • 위치 정보
  • 연락처, 사진, 파일 또는 마이크 데이터

Bluetooth

TouchPilot은 본인의 iPad가 iPhone의 키보드와 포인터 입력을 받을 수 있도록 Bluetooth를 사용합니다. Bluetooth 권한은 앱의 입력 기기 동작에만 사용됩니다. 앱은 Bluetooth 데이터를 ScratchWell 서버로 보내지 않습니다.

입력 데이터

트랙패드, 키보드, 기능 키 또는 단축키를 사용할 때 TouchPilot은 연결된 iPad로 해당 입력 보고서를 보냅니다.

TouchPilot은 사용자의 입력 텍스트나 키 이벤트를 ScratchWell로 보내지 않습니다. 현재 앱에는 개발자가 운영하는 서버가 없습니다.

iPhone에 저장되는 설정

TouchPilot은 Apple의 시스템 저장소를 사용해 설정을 iPhone에 로컬로 저장합니다. 저장될 수 있는 설정은 다음과 같습니다.

  • 트랙패드 감도와 가속
  • 스크롤 레일 위치와 스크롤 방향
  • 드래그 타이밍
  • 햅틱 피드백 설정
  • 단축키 정의
  • 튜토리얼 완료 상태
  • Universal Clipboard 설정

이 설정은 기기에 저장되며 ScratchWell로 전송되지 않습니다.

Universal Clipboard

이모지처럼 표준 Bluetooth 키보드 보고서로 직접 보낼 수 없는 문자가 있습니다. Universal Clipboard 대체 전송을 사용하도록 선택하면 TouchPilot은 해당 텍스트를 iPhone 클립보드에 넣고 연결된 iPad로 붙여넣기 명령을 보낼 수 있습니다.

Universal Clipboard는 Apple 시스템 기능입니다. 같은 Apple Account, Handoff, Bluetooth, Wi-Fi 및 가까운 기기가 필요합니다. TouchPilot은 클립보드 전송 성공 여부를 확인할 수 없습니다.

백그라운드 동작

TouchPilot은 앱이 백그라운드로 이동하면 광고, 스캔 및 입력 전송을 중지합니다. 앱은 열려 있고 활성 상태일 때 사용하는 것을 기준으로 합니다.

제3자 서비스

TouchPilot은 현재 제3자 분석, 광고 또는 크래시 리포팅 SDK를 포함하지 않습니다. 향후 변경되는 경우 출시 전에 이 방침과 App Store 개인정보 세부 정보를 업데이트합니다.

아동

TouchPilot은 생산성 유틸리티이며 아동을 대상으로 하지 않습니다. 앱은 아동의 개인정보를 고의로 수집하지 않습니다.

방침 변경

ScratchWell은 TouchPilot 변경에 따라 이 방침을 업데이트할 수 있습니다. 방침이 변경되면 시행일을 업데이트합니다.
