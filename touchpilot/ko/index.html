TouchPilot 지원

TouchPilot은 iPhone을 iPad용 가벼운 트랙패드와 키보드 입력 화면으로 바꿔주는 앱입니다.

지원 기능

  • Bluetooth를 통한 iPhone-to-iPad 포인터 입력
  • 클릭, 우클릭, 드래그 유지, 가장자리 스크롤
  • 시스템 키보드, 기능 키, 사용자 단축키
  • 라이트 모드와 다크 모드

중요 안내

  • TouchPilot v1은 iPad를 수신 기기로 사용하는 것을 기준으로 설계되었습니다.
  • Mac과 일반 Bluetooth 기기는 현재 지원하지 않습니다.
  • 앱이 백그라운드로 이동하면 입력 전송을 중지합니다.
  • 실제 호환성은 기기와 OS 버전에 따라 달라질 수 있습니다.

기본 연결 방법

  1. iPhone에서 TouchPilot을 엽니다.
  2. Bluetooth 접근 권한을 허용합니다.
  3. TouchPilot을 열어둔 상태로 iPad 연결 준비가 자동으로 시작되게 합니다.
  4. iPad에서 설정 > Bluetooth를 엽니다.
  5. TouchPilot 항목이 보이면 선택합니다. iPadOS에서는 일반 Bluetooth 기기 또는 액세서리로 보일 수도 있습니다.

문제 해결

  • iPad에서 TouchPilot이 보이지 않으면 두 기기의 Bluetooth를 껐다가 다시 켭니다.
  • 이전 페어링이 남아 있으면 iPad Bluetooth 설정에서 기존 TouchPilot 또는 일반 액세서리 항목을 지운 뒤 다시 연결합니다.
  • 사용 중에는 TouchPilot을 foreground에서 열어 둡니다.
